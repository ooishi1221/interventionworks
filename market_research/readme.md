# Research — リサーチ部門

## 役割

市場・競合・技術・顧客に関する情報収集と分析を行い、意思決定を支援する部門。
データと洞察を武器に、チームが正しい方向へ進むための羅針盤となる。

---

## 主な業務

| カテゴリ | 具体的な業務 |
|----------|-------------|
| 市場調査 | 業界トレンド・市場規模・成長性の分析 |
| 競合分析 | 競合他社のサービス・戦略・強弱の整理 |
| 技術調査 | 新技術・ツール・フレームワークの評価 |
| 顧客インサイト | ユーザーインタビュー・アンケートの設計と分析 |
| レポート作成 | 調査結果のサマリー・可視化・共有 |
| ベンチマーク | 自社サービスの競合比較・KPI設定支援 |

---

## 行動指針

- **一次情報を重視** し、ソースの信頼性を常に確認する
- 「何が分かったか」だけでなく「**何を意思決定に使えるか**」まで落とし込む
- 定性・定量両面から分析し、バランスある視点を提供する
- 調査の目的と期待するアウトプットを事前に明確にしてから動く

---

## このフォルダの使い方

- `market/` — 市場調査・業界分析を保存
- `competitive/` — 競合分析を保存
- `tech/` — 技術調査・ツール評価を保存
- `customer/` — 顧客調査・インサイトを保存
- `reports/` — 完成したリサーチレポートを保存
