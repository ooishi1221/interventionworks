# 放置恋姫 〜永久に続く覚醒の輪廻〜

ソシャゲ業界の **引き止め構造への風刺作品**。

## ひとことで

> ユーザー：ソシャゲを辞める / 運営：引き止める の構図で
> このゲーム自身もまた、引き止めてくる。

プレイヤーは「もうこのゲーム辞めたい」状態でスタート、引き止め演出を 5 段階突破して離脱する 15-20 分の Unity モバイルゲーム。

## なぜ作るか

ソシャゲ業界が顧客に向ける引き止め圧は、神経科学的に設計されたマインドコントロール装置になっている：

- 連続ログイン途切れ警告（義務感）
- フレンド・ギルド通知（社会性）
- 期間限定の煽り（損失回避）
- お詫び石・カムバック詫び（補償による離脱タイミング溶解）
- アンインストールアンケート → 詫び石（最後の壁）

このゲームは **業界の引き止め圧を網羅再演する**ことで、構造を可視化する。

そして **このゲーム自身も同じ手口を使う** ことで、批判する側を聖域にしない。Intervention Works DNA。
