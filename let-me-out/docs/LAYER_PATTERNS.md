# Layer Patterns — 引き止めパターン文言プール

> 2026-05-03 起こし。各 Layer 内の引き止め文言プール。実装時にここからランダム抽出して 1 セッションに 3 個出す（Layer ごとに）。

## 6 カテゴリ × symbolize

| カテゴリ | パターン | 業界の symbolize |
|---|---|---|
| 義務感 | 連続ログイン途切れ警告 | 義務化されたエンタメ |
| 社会性 | フレンド・ギルド通知 | 人間関係の人質 |
| 損失回避 | 限定・期間 | 焦らせの定型 |
| 補償 | 詫び石・アンケ詫び | 引退タイミングを溶かす |
| サンクコスト | 「あなたの 847 日」 | 過去への固定 |
| ストア最適化 | 評価依頼ポップアップ | 退会時に星 5 を狙う厚顔 |

## Layer 5: アンケート引き止め（最後の壁）

### 文言（10 問の選択式）

1. ご利用期間を教えてください
2. 最も楽しんだコンテンツは？（[]ガチャ []ストーリー []育成 []ギルド []イベント）
3. 改善してほしい点は？（[]ガチャ確率 []ストーリー []運営対応 []演出 []課金導線）
4. 他のゲームに移行されますか？（[]はい、タイトルを記入 []いいえ）
5. 復帰の可能性は？（[]あり []条件次第 []なし）
6. ガチャの天井設定について（[]適切 []高い []低い）
7. ストーリーの満足度（5 段階）
8. 課金額の妥当性（5 段階）
9. 運営対応の評価（5 段階）
10. ご意見・ご感想（自由記述）

## 文言追加方針

- 各 Layer **最低 10 個**になるまで追加していく（現在の draft は 5-10 個）
- 業界 insider（裕司）が「あ、これあるw」と笑う粒度を維持
- 業界 outsider にも「気持ち悪さ」が伝わる範囲で
- 中華 ローカライズ風と日本系の文言を **混在**させる（aesthetic フランケン構造に整合）

## 実装メモ

### ランダム抽出ロジック（Unity 実装）

```
Layer N の文言プール → 全 10 個から 3 個ランダム選択 → 順序ランダム
```

セッション内で同じ文言を 2 回出さない（uniqueness 保証）。
