## 確定パラメータ

| 項目 | 値 |
|---|---|
| プレイ時間 | 15-20 分 / 1 セッション |
| Layer 構成 | 固定順序（1→5）+ 各 Layer 内文言ランダム抽出 |
| メタ構造 | クリア直後 + 2 回目起動時の二重メタ |
| プラットフォーム | Unity / iOS + Android |
| aesthetic | 中華放置 RPG 風 × 日本ガチャ演出 |
| マネタイズ | 完全無料（IAP なし）|

## 1 セッションの時系列フロー

### Layer 0: 起動画面

> 「お疲れ様でした。`放置恋姫`はサービス開始から 3 年、あなたを 847 日お迎えしました。本日アンインストールをご検討いただき、誠にありがとうございます。最後に少しだけお時間いただけますか？」

「アプリを閉じる」ボタンを押すとプレイ開始。

### Layer 1: 日常引き止め（義務感）

- 「ログインボーナス受け取りますか？」
- 未受取消滅赤字警告
- 「7 日連続まであと 1 日です！」連続ボーナス更新

### Layer 2: 人間関係引き止め（社会性）

- 「ギルドメンバー *Tanaka* さんがあなたに ❤️ を送りました」
- 「フレンドの *Yuji* さんがログインしました（最終 6 ヶ月ぶり）」
- フェード長め、BGM しっとり

### Layer 3: 機会損失引き止め（損失回避）

- 「【あと 23 時間】100 連無料ガチャ開催中」
- 「あなたが集めていた限定 SSR 復刻」
- タイムリミット常時点滅

### Layer 4: 詫び引き止め（補償）

- 「【お知らせ】先週のメンテにご迷惑をおかけしました。お詫び石 3,000 個進呈」
- 受け取ると残高表示で**ガチャ画面に強制遷移**

### Layer 5: アンケート引き止め（最後の壁）

- 「アンインストール」→ 長文アンケート（10 問・選択式）
- 「ご回答ありがとうございました。お詫び石 5,000 個」
- 残高更新演出 → ガチャに戻る

### 勝利

- 実際に閉じられた瞬間 → 「**Congratulations. You are free.**」

## スコア設計

| 指標 | 値 |
|---|---|
| 離脱までの所要時間 | 短いほど勝ち |
| 食らった引き止め数 | 少ないほど勝ち |
| 詫び石残高 | **多いほど"敗北"スコア**（持って帰れない無価値資源）|
| 罪悪感メーター | "ギルドのみんなを見捨てる" を選ぶと加算 |

最終画面：

> 「あなたは 23 分 47 秒で自由になりました。詫び石 47,000 個はこの世界に置いていきます。」

## メタ構造（風刺の二重化）

### 1. クリア直後の 1 撃メタ

「You are free」の直後、アプリ閉じようとすると **"放置恋姫" 自身が**「アンケートにご協力ください」と出す。

→ 答えると詫び石（無価値）→ ホーム画面に戻される。

### 2. 2 回目起動時のカムバックメタ

「お帰りなさい！カムバックキャンペーン詫び石 10,000 個進呈」

→ 業界の離脱者追跡 CRM を逆手に取る。

### メッセージ

> 業界を風刺してるはずのこのゲームも、結局同じ手口でお前を引き止める。
> 批判する側も逃げられない構造の中にいる。

→ Intervention Works DNA：批判者を聖域にしない（Slight の「ダサくない安全」と同じ自己整合性）。

## 開発フェーズ詳細

### Phase 1: POC（2-3 週間）

**目的:** 「業界アプリじゃん」体感が成立するか確認

実装範囲：
- 起動画面（Layer 0）
- Layer 3（限定ガチャ煽り）の演出フル実装
- 中華 UI 過密度のサンプル
- ガチャ演出 1 種（SSR 確定光）

判定基準：
- 業界 insider（裕司）が「これ業界アプリじゃん」と錯覚する
- 業界 outsider が**戸惑わずに引き止め圧を感じる**

### Phase 2: フル実装（3-4 ヶ月）

- 5 Layer + メタ構造
- iOS / Android ビルド
- ストア審査トライ
- TestFlight → 内部テスター → 一般公開

### Phase 3: ストア審査ドキュメント化

- 審査経緯（通過 / ハネられた）を記録
- 通過 → 「審査も含めてプロダクトです」
- ハネられた → 「業界の構造に阻まれて出せませんでした」を Twitter / note 拡散

## 法的境界

- ❌ 特定タイトル名・キャラそのままは NG
- ⭕ 業界共通の手口・archetype・aesthetic だけ inspired
- ⚠️ release 前に弁護士相談必須

### App Store / Google Play 審査リスク

| リスク | 度合い | 対策 |
|---|---|---|
| Dark Pattern 規約抵触 | 中〜高 | "satire / parody" 明示、Indie Game 枠、TestFlight 先行 |
| 詫び石が IAP 規約に見える | 中 | 完全無料・購入導線なしで明確化 |
| 既存タイトルパロディ判定 | 低〜中 | 「業界共通の手口」風刺に絞る、特定タイトル想起させない |
