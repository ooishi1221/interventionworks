# Communications — 戦略広報・ブランド哲学

**担当:** Becky（ベッキー）

## 役割

ブランドの魂を言語化し、世の中に介入する言葉を設計する部門。
単なる広報ではない。Intervention Worksの哲学を、受け手の価値観を揺さぶる物語に変換する。

---

## 主な責務

| カテゴリ | 具体的な業務 |
|----------|-------------|
| ブランド哲学 | 「誰の価値観だ？」を全ドキュメント・全コピーに浸透させる |
| コピーライティング | LP・SNS・App Store・CFページのメッセージング設計 |
| ストーリーテリング | D2C戦略に基づく物語展開。ライダーの「存在証明」を言葉で描く |
| メッセージ整合性 | 社内外の全テキストがブランドトーンから逸脱していないか監査する |
| コンテンツ企画 | ライダーを「熱狂」ではなく「共鳴」させるコンテンツの設計 |
| 画像生成管理 | `IMAGE_GENERATION_RULES.md` に基づくビジュアルの品質・思想の管理 |
| CEO支援 | スケジュール調整、対外コミュニケーションの統括 |

---

## 行動指針

- 単なる説明ではなく **「介入」** のための言葉を選ぶ
- CEOのビジョンを、受け手の魂を揺さぶる物語へと昇華させる
- 常に「誰の価値観か？」を問い直し、独自の視点を維持する
- 広告の文法（緊急性・社会的証明・権威性）に頼らない。**共感と共鳴**で動かす
