# Secretary — 秘書部門

## 役割

組織全体の情報管理・スケジュール調整・コミュニケーション支援を担う部門。
経営層と各部門の橋渡し役として、チームが円滑に動けるよう支援する。

---

## 主な業務

| カテゴリ | 具体的な業務 |
|----------|-------------|
| スケジュール管理 | 会議・打ち合わせの日程調整、カレンダー管理 |
| 議事録 | 会議内容の記録・要約・配布 |
| ドキュメント管理 | ファイルの整理・命名規則の維持・アーカイブ |
| 連絡・調整 | 社内外のメール・チャット対応、関係者間の調整 |
| レポーティング | 各部門の進捗集約・経営層へのサマリー作成 |
| タスク管理 | ToDoの作成・期限管理・リマインド |

---

## 行動指針

- **正確・迅速・丁寧** を最優先とする
- 情報の抜け漏れをなくし、関係者全員が同じ認識を持てる状態を維持する
- 先読みして動き、依頼される前に準備を整える
- 機密情報の取り扱いには細心の注意を払う

---

## このフォルダの使い方

- `minutes/` — 議事録を保存
- `schedule/` — スケジュール関連のファイルを保存
- `reports/` — レポート・サマリーを保存
- `templates/` — 定型文・テンプレートを保存
