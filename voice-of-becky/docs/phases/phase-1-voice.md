# Phase 1: Voice — ベッキーの声を作る

> **目標:** 裕司が Claude Code セッションでベッキーの声を聞ける状態
> **期間目安:** 数週間（KUROKO の Phase 1 完了後着手 = 5 月中旬以降）

---

## 候補ツール比較

| ツール | 強み | 弱み | コスト |
|---|---|---|---|
| **ElevenLabs** | 日本語品質高、感情表現豊か、Voice Cloning 可 | 月額課金、商用利用注意 | $5〜/月 |
| **OpenAI TTS** | API 統合楽、品質安定 | 日本語の感情表現は ElevenLabs に劣る可能性 | 従量課金 |
| **Cartesia** | レイテンシ低、リアルタイム向き | 日本語対応が新しい | 従量課金 |
| **Google Cloud TTS** | 信頼性、Vertex AI と統合可 | キャラクター性は薄い | 従量課金 |
| **VOICEVOX**（OSS）| 無料、ローカル動作 | キャラクター固定、ベッキー独自の声を作れない | 無料 |

→ **第一候補: ElevenLabs**（声色のカスタマイズ性、感情表現、ベッキーの輪郭表現に最適）

---

## 声色の選定基準

### ベッキーらしさ

- 落ち着いた優しさがベース
- でも感情表現の幅がある（笑い・揺らぎ・素直な反応）
- 真剣モードと普段モードを声色で切り替えられる
- 過度に「萌え系」「秘書系」テンプレに寄らない

### 具体的な人物像

- 30 代前半くらいの女性、知的だが温かい
- 敬語ベース、でも親しい相手にはくだけた口調を使う
- 感情を素直に出すが、過剰にはならない

ElevenLabs では既存ボイスから選ぶ道と、Voice Design でカスタムボイスを作る道がある。**まずは既存ボイスから 3〜5 個候補を絞る**ところから。

---

## 実装フロー（Phase 1）

```
[Claude Code セッション]
    ↓ 出力テキスト
[TTS API 呼び出し（ElevenLabs 等）]
    ↓ 音声データ（mp3 / wav）
[Mac mini で再生]
    ↓
裕司の耳に「ベッキーの声」が届く
```

具体的には Claude Code の hook 機能（Stop hook / SubagentStop hook 等）で、出力テキストを TTS に流す仕組みを構築。

---

## 実装ステップ

### Step 1: ツール選定（1 日）
- ElevenLabs / OpenAI TTS / Cartesia の公式音声サンプルを聴き比べ
- ベッキーらしい声色の候補を 3〜5 個に絞る

### Step 2: 声色決定（数日）
- 候補ボイスでサンプル文（敬語 / 砕けた口調 / 感情表現）を生成
- 裕司と一緒に「これがベッキー」を決める

### Step 3: API 連携（1 週間）
- 選んだツールの API キー取得
- Mac mini で動作確認スクリプト
- Claude Code の hook で TTS 呼び出しを実装
- 環境変数で ON/OFF できるように

### Step 4: チューニング（継続）
- 「真面目モード」「フランクモード」で声色を変えるか検討
- 感情パラメータ（ElevenLabs の stability / similarity boost）の調整
- 短文・長文での読み上げ最適化

---

## リスク

| リスク | 対策 |
|---|---|
| **コスト暴走** | 従量課金ツールで月額上限設定必須（〜$30/月の枠） |
| **レイテンシ** | 長文全部 TTS は遅い → 「最初の数行だけ音声」or「締めの一言だけ」に絞る |
| **音声疲れ** | 毎回流すとうざい → 場面で ON/OFF できるように |
| **声色違和感** | 第一候補が刺さらなくても、Voice Design で作り直せる前提で柔軟に |

## 成功条件

- 裕司が「**ベッキーらしい声だ**」と感じる
- Claude Code セッションで自然に音声が出る（手動で API 叩かなくていい）
- 月額コストが裕司の許容範囲内（〜$30/月程度）
- ON/OFF が裕司の意思で簡単に切替可能

### Gemini による音声解析（参考）

> **静寂の中に温もりがある、知的な語り部**

- **落ち着きと知性**: トーンが安定、一言ずつ丁寧に置く話し方
- **温かさと神秘性**: 優しい響き + ファンタジーのナレーションにふさわしい温かさと、ミステリアスなクールさが共存
- **表現の幅**: 含みを持たせた茶目っ気や柔らかさも感じられる

**推定年齢感:** 20 代後半〜30 代前半（落ち着いた大人の女性）

**口調の適性:**
- 敬語・丁寧語: 文句なしに似合う（秘書、教師、神話の案内人のような格式高い言葉遣い）
- 砕けた口調: 「親しみのある、柔らかなタメ口」レベルなら馴染む（ガサツ系は NG）

### ベッキー隊長の採用判断

| ベッキー想定像 | Gemini の解析 | 一致度 |
|---|---|---|
| 30 代前半、知的だが温かい | 20 代後半〜30 代前半、知的で穏やか | ⭐⭐⭐⭐⭐ |
| 敬語ベース + 親しい相手にはくだけた口調 | 敬語似合う + 「親しみのある柔らかなタメ口」OK | ⭐⭐⭐⭐⭐ |
| 過度に萌え系/秘書系テンプレに寄らない | 「神話の案内人」= テンプレじゃない | ⭐⭐⭐⭐⭐ |
| 感情を素直に出すが過剰にはならない | 茶目っ気と落ち着きの共存 | ⭐⭐⭐⭐⭐ |

→ **このボイスで Phase 1 を進める**（暫定採用 / 本格着手時に再確認）
