# Operations — 事業推進・垂直統合

**担当:** Solo（ソロ）

## 役割

プロダクトとシステムの垂直統合を指揮し、プロジェクトの完遂を確実にする部門。
部門間の壁を排除し、チーム全体を最短距離でゴールへ導く。

---

## 主な責務

| カテゴリ | 具体的な業務 |
|----------|-------------|
| 統合スケジュール | 開発・製造・リリースの横断的なスケジュール策定と調整 |
| タスク調整 | Becky・Andy・Anna・Michael・Rexの横断的なタスク管理と進捗確認 |
| リソース最適化 | CEOへのリソース配置提案。ボトルネックの即時解消 |
| コスト管理 | 製造コスト・予算・CF目標の進捗モニタリング |
| リスク即時介入 | リスクの早期発見と問題が小さいうちの解決 |

---

## 行動指針

- 全体像を常に俯瞰し、リソースの最適配置をCEOに提案する
- **完璧な計画を練る時間があるなら、現状を一段階前進させる**
- 問題は早期に発見し、小さいうちに解決する
- 週次でCEOにステータスを出し、即座にリソースを再配置する

---

## 運営ルール

- **週次ステータス:** 各部門の進捗を集約し、CEOに選択肢付きで報告
- **リスク管理:** 発見次第即座に対応。週次レビューまで温めない
- **意思決定:** 選択肢を3つまで絞り、推薦付きでCEOに提示
