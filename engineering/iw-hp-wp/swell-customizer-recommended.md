# SWELL カスタマイザー推奨値（IW HP 5/27 第一弾）

> 外観 → カスタマイズ で全部いじれる。各項目「推奨値」と「なぜ」を併記。

## 1. サイト全体設定

| 項目 | 推奨値 | 理由 |
|---|---|---|
| サイト基本色 | `#000000`（black） | 引き算の美学・反乱の世界観 |
| メインカラー | `#000000` | 同上 |
| テキストカラー | `#1a1a1a` | 真っ黒は目疲れ、わずかにオフ |
| リンクカラー | `#000000` + 下線 | 装飾せず機能だけ |
| サイト幅 | `1100px` | 読み物として広すぎず |

## 2. ヘッダー

| 項目 | 推奨値 |
|---|---|
| ヘッダーレイアウト | センターロゴ |
| 追従ヘッダー | OFF（スクロール邪魔）|
| ヘッダー背景 | 透過 → スクロールで white |

## 3. SEO SET（最重要）

```
SWELL設定 → SEO設定
```

| 項目 | 設定値 |
|---|---|
| サイト名（og:site_name） | `Intervention Works` |
| デフォルト description | `既存の価値観に介入する。Wit-One が運営する屋号、AI とプロダクトで世界に反乱する場所。` |
| デフォルト OG 画像 | `iw-ogp-default.png`（1200x630, 後でベッキー素材） |
| Twitter Card | `summary_large_image` |
| Twitter @username | （裕司の X アカウント決まり次第）|

## 4. フッター

| 項目 | 設定値 |
|---|---|
| フッター背景色 | `#000000` |
| フッター文字色 | `#ffffff` |
| コピーライト | `© Intervention Works` |
| ウィジェット | note リンク / X リンク / プライバシー / お問い合わせ |

## 5. パフォーマンス（SWELL設定 → 高速化）

| 項目 | 推奨 |
|---|---|
| ページ遷移高速化 | ON（プリフェッチ）|
| 遅延読み込み | 全画像 ON |
| キャッシュ機能 | ON |
| CSS インライン化 | ON（FCP 改善）|

## 6. やらないこと（5/27 第一弾）

- カラー設計の作り込み（ブランドガイド固まってから）
- 細かいタイポグラフィ調整（コピー入った後で見ながら）
- アニメーション設定（パフォーマンス計測してから）
