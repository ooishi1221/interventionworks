# ムームーサーバー（GMO ペパボ）固有メモ

> ムームーサーバー = ロリポップ! 系の派生。基本仕様はロリポップ準拠。

## 1. PHP バージョン確認

**確認手順:**
1. ムームーサーバー管理画面ログイン
2. 「PHP 設定」or「サーバー設定」→ PHP バージョン確認
3. **推奨: PHP 8.2 以上**（SWELL 公式要件 PHP 7.4 以上だが、8.2 で速度 2 倍）

**注意:** PHP バージョン変更すると WP プラグイン互換性問題が出る場合あり。**5/27 まで触らない**、もし PHP 7.x なら 5/28 以降に上げる。

## 2. SSL 証明書

ムームーサーバーは **Let's Encrypt 無料 SSL** が標準。

**確認:** https://intervention.jp/ で鍵マークが出てれば OK。出てない場合は管理画面 → 「独自 SSL 証明書導入」→ Let's Encrypt 設定。

## 3. PHP メモリ制限

ムームーサーバー（ロリポップ系）の default は `128M`。SWELL は `256M` 推奨。

**確認方法:** WP 管理画面 → ツール → サイトヘルス → 情報 → 「サーバー」→ PHP メモリ制限

**256M 未満だった場合の対応:**
- ムームーサーバー管理画面 → 「PHP 設定」→ memory_limit を `256M` に変更
- それでダメなら `wp-config.php` に追加: `define('WP_MEMORY_LIMIT', '256M');`

## 4. アップロードファイルサイズ上限

ムームーサーバー default `100MB`。OGP 画像（数 MB）レベルなら問題なし。動画上げる予定があれば確認。

## 5. .htaccess（http → https リダイレクト）

WP のサイトアドレスを https に統一しても、http 直アクセスがリダイレクトされない場合は `.htaccess` に追記:

```apache
<IfModule mod_rewrite.c>
RewriteEngine On
RewriteCond %{HTTPS} off
RewriteRule ^(.*)$ https://%{HTTP_HOST}%{REQUEST_URI} [R=301,L]
</IfModule>
```

**やる場所:** ムームーサーバー管理画面 → 「ロリポップ! FTP」→ ルートの `.htaccess` を編集
**注意:** 編集前に必ずバックアップ（FTP でダウンロード）。

## 6. バックアップ

ムームーサーバーは **「7 世代バックアップ」が有料オプション**（月 330 円）。

**推奨:** 5/27 第一弾の前に**手動バックアップを 1 回取る**。
- WP 管理画面 → ツール → エクスポート（コンテンツ）
- ムームーサーバー FTP で `/wp-content/` 全体ダウンロード（テーマ + アップロード画像）
- DB は phpMyAdmin で SQL エクスポート

**自動化したいなら:** プラグイン「UpdraftPlus」を入れて Google Drive に週次バックアップ（無料版で OK）。

## 7. 既知の地雷

- ムームーサーバーは `wp-cron` の自動実行が不安定 → 予約投稿が動かないことあり
- 対応: プラグイン「WP Crontrol」で手動実行確認、or サーバー側 cron に置き換え（5/27 第一弾では不要）
