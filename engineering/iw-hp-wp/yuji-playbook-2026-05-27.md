## a. URL を https に統一（5 分）

**やること:** site URL の http/https 混在を直す。

1. 管理画面左メニュー → **設定 → 一般**
2. 以下 2 項目が両方 `https://intervention.jp`（**末尾スラッシュなし**）になってるか確認
   - WordPress アドレス (URL)
   - サイトアドレス (URL)
3. 違ってたら直して、ページ最下部の **「変更を保存」**

**確認:** ブラウザで `http://intervention.jp` を開く → 自動で `https://` にリダイレクトされれば OK。鍵マークが表示されてればさらに OK。

## b. SWELL の SEO SET 設定（10 分）

**やること:** OGP（SNS シェア時のサムネ）、description（検索結果の説明文）、Twitter Card を設定。

1. 左メニュー → **SWELL設定 → SEO設定**
2. 以下を入力（コピペ用テキストは `swell-customizer-recommended.md` の §3 参照）:

   - **デフォルト description:** `既存の価値観に介入する。Wit-One が運営する屋号、AI とプロダクトで世界に反乱する場所。`
   - **デフォルト OG 画像:** （後述 e で差し替えた `iw-ogp-default.png` を選択）
   - **Twitter Card タイプ:** `summary_large_image`
3. ページ下部 **「変更を保存」**

**確認:** https://www.opengraph.xyz/url/https%3A%2F%2Fintervention.jp に貼って、画像 + タイトル + 説明文が出れば OK。

## c. note RSS の埋め込み（5 分）

**やること:** home page の「Journal」セクションに note 最新記事を自動表示。

**推奨方法: SWELL ブロックエディタ標準の「外部記事」or 「投稿リスト」ブロック使わず、WP 標準「RSS」ブロック を使う**（プラグイン追加不要・最軽量）。

1. 左メニュー → **固定ページ → home（フロントページ）→ 編集**
2. 「Journal」or「Voice」と書きたい箇所にカーソル → **+ ボタン → 「RSS」で検索 → RSS ブロック追加**
3. URL 欄に貼り付け: `https://note.com/intervention_jp/rss`
4. 「使用」をクリック
5. ブロック右側の設定パネルで:
   - **項目数:** `3`
   - **日付を表示:** ON
   - **抜粋を表示:** ON
   - **抜粋の最大文字数:** `100`
6. 右上 **「更新」**

**確認:** home page を別タブで開いて、note の最新 3 本タイトルが表示されてれば OK。

> プラグイン「WP RSS Aggregator」は不要（WP 標準 RSS ブロックで足りる）。重くなるだけ。

## d. フッターに note / X 導線追加（5 分）

**やること:** フッターメニューに note と X (Twitter) のリンクを追加。

1. 左メニュー → **外観 → メニュー**
2. 上部「編集するメニューを選択」で **「フッターメニュー」を選択**（なければ「新規メニュー作成」→ 名前 `フッターメニュー` → メニュー位置で「フッター」にチェック）
3. 左カラム **「カスタムリンク」** を開く
4. note 追加:
   - URL: `https://note.com/intervention_jp`
   - リンク文字列: `note`
   - **「メニューに追加」**
5. X 追加:
   - URL: `https://x.com/<裕司の X ハンドル>` ← **ハンドル確定後に修正**
   - リンク文字列: `X`
   - **「メニューに追加」**
6. ページ下部 **「メニューを保存」**

**確認:** home page 下にスクロール → フッターに note / X が出ていて、クリックで別タブ遷移すれば OK。

> X アカウントまだ仮なら、リンク文字列だけ `X (準備中)` で URL は `#` にしておいて、後日差し替えで OK。

## e. プレースホルダ画像 7 枚の差し替え（5〜10 分／枚数次第）

**やること:** ベッキーから渡される画像 7 枚を、WP メディアライブラリに上げてブロックエディタで差し替える。

### e-1. アップロード

1. 左メニュー → **メディア → 新規追加**
2. ベッキーから受け取った画像（7 枚）をドラッグ&ドロップ
3. 全部「アップロード完了」になるまで待つ
4. 各画像をクリック → 右パネルの **「代替テキスト」** に画像内容を 1 行で書く（例: `Intervention Works ロゴ — 黒地に白文字`）
   - SEO + アクセシビリティの両方に効く

### e-2. ブロックエディタで差し替え

1. 左メニュー → **固定ページ → home → 編集**
2. 差し替えたい画像ブロックをクリック
3. ブロック左上のツールバー → **「画像を置換」** → 「メディアライブラリから選択」
4. アップロードした画像を選択 → **「置換」**
5. 全 7 枚やり終えたら、右上 **「更新」**

**確認:** home page を別タブで「強制リロード」（Mac: `Cmd+Shift+R`、Win: `Ctrl+F5`）→ 全部新しい画像になってれば OK。

> ベッキーから画像が届いてない時点で、この e タスクはスキップして OK。a〜d だけで第一弾の最低ラインは満たす。
