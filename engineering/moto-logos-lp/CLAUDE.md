# Moto-Logos LP（ランディングページ）

## 概要

**公開URL:** https://moto-logos.web.app  
**目的:** Moto-Logos アプリの世界観（存在証明）を伝え、ダウンロードへ導くランディングページ。

## コンセプト転換に伴う方針（2026-04-14）

LP全体のコピー・メッセージングを「存在証明」コンセプトに合わせて全面書き直し予定。

### 変更の方向性
- 旧キャッチコピー「あなたの一報がマップに命を灯す」→ 「存在証明」「足跡」「気配」をキーワードに刷新
- 「報告」「貢献」「共有」→ 「メモ」「足跡」「ここにいたよ」に言い換え
- 善意ベースの訴求 → 利己→結果的利他の訴求に転換
- 「ユーザーの匂い」「体温のある情報」= Googleマップとの差別化を明確に打ち出す
- ライダーの少数派感・孤独感 → Moto-Logosで「仲間がいる気配」を感じられるストーリー

```bash

```

## トーン & メッセージング

- キャッチコピー方向性: 「自分のメモが、誰かの安心になる。」「ここにいたよ、を残そう。」（最終稿は要Beckyレビュー）
- ライダー同士の「気配」で語る。強い繋がりは強要しない。ヤエー（ピースサイン）の距離感
- 「報告してください」「貢献しよう」とは言わない
- 機能説明より「存在の実感」を描写する
- 比較広告的な表現は避け、自分たちの世界観で語る
- Googleマップとの差別化: データの正確さではなく「ユーザーの匂い」「体温のある情報」
