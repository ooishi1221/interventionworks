# Moto-Logos LP（ランディングページ）

## 概要

**公開URL:** https://moto-logos.web.app  
**目的:** Moto-Logos アプリのコンセプトを伝え、ダウンロードへ導くランディングページ。

## 技術スタック

| レイヤー | 技術 |
|---------|------|
| フレームワーク | Vite 8 + React 19 |
| 言語 | TypeScript |
| ホスティング | Firebase Hosting（プロジェクト: `moto-spotter`、サイト: `moto-logos`） |
| フォント | Noto Sans JP（本文）、Inter（数値） |

## デプロイ

```bash
npm run build && firebase deploy --only hosting --project moto-spotter
```

## ページ構成（9セクション）

| # | セクション | コンポーネント | 内容 |
|---|-----------|---------------|------|
| 1 | ヒーロー | `Hero.tsx` | キャッチコピー + アプリモック + DLボタン |
| 2 | ペイン | `Pain.tsx` | 3つの課題カード + 赤い鮮度バッジ |
| 3 | コアバリュー | `CoreValues.tsx` | グローブ操作 / 鮮度バッジスライダー / マップ増殖 |
| 4 | ライブ統計 | `Stats.tsx` | カウンター3つ + アクティビティフィード |
| 5 | テスティモニアル | `Testimonials.tsx` | ライダー4名の声 |
| 6 | 使い方 | `HowTo.tsx` | 3ステップ（DL → 開く → 報告） |
| 7 | FAQ | `Faq.tsx` | 5問（開閉アコーディオン） |
| 8 | 最終CTA | `FinalCta.tsx` | 「仲間と、地図を育てよう。」 |
| 9 | フッター | `Footer.tsx` | コピーライト |
| - | スティッキーCTA | `StickyCta.tsx` | 画面下部固定のDLボタン |

## トーン & メッセージング

- キャッチコピー: 「一人の発見を、全ライダーの安心に。」
- ライダー同士の「仲間意識」で語る。上から教えるのではなく、一緒に地図を作る感覚
- 機能説明より「体験」を描写する
- 比較広告的な表現は避け、自分たちの世界観で語る

- コアバリュー3のマップ増殖 → アニメーションに差し替え
