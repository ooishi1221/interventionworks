## 1. はじめに

Intervention Works（以下「当社」といいます）は、モバイルアプリケーション「Moto-Logos」（モトロゴス、以下「本アプリ」といいます）をご利用いただくユーザーの皆さま（以下「ユーザー」といいます）のプライバシーを尊重し、個人情報の保護に関する法律（個人情報保護法）その他の関連法令を遵守いたします。

本プライバシーポリシー（以下「本ポリシー」といいます）は、当社が本アプリを通じて取得する情報の種類、利用目的、管理方法等について定めるものです。

## 3. 取得する情報

当社は、本アプリの提供にあたり、以下の情報を取得します。

### 3.1 位置情報（GPS）

| 項目 | 内容 |
|------|------|
| **取得する情報** | 端末のGPSから取得する緯度・経度の情報 |
| **取得方法** | OSの位置情報許可ダイアログを通じた、ユーザーの明示的な許可（オプトイン） |
| **利用目的** | 周辺の駐車スポットの検索・表示、地図上でのユーザー現在地の表示 |
| **保存場所** | 端末内のみ（サーバーには送信されません） |
| **任意/必須** | 任意（位置情報を許可しなくてもアプリの基本機能は利用可能ですが、周辺スポット検索等の一部機能が制限されます） |

### 3.2 デバイスID（匿名UUID）

| 項目 | 内容 |
|------|------|
| **取得する情報** | アプリ初回起動時に端末内で自動生成される匿名の一意識別子（UUID） |
| **取得方法** | アプリにより自動生成し、端末内のAsyncStorageに保存 |
| **利用目的** | アプリの利用状況の統計分析（DAU/WAU/MAU等のアクティブユーザー数の集計） |
| **保存場所** | 端末内のAsyncStorage、およびFirestoreのuser_activityコレクション |
| **任意/必須** | 自動（アプリ利用に伴い自動的に生成されます） |

**注意:** デバイスIDは、ユーザーの氏名、メールアドレス、電話番号等の個人を直接特定できる情報とは紐付けられません。

### 3.3 ニックネーム

| 項目 | 内容 |
|------|------|
| **取得する情報** | ユーザーが任意に設定する表示名 |
| **取得方法** | ユーザーによるアプリ内での入力 |
| **利用目的** | レビュー投稿時の投稿者名の表示 |
| **保存場所** | 端末内のAsyncStorage、およびレビュー投稿時にFirestoreのレビューデータに含まれる |
| **任意/必須** | 任意（設定しなくてもアプリの利用は可能です） |

### 3.4 ユーザー投稿コンテンツ（UGC）

| 項目 | 内容 |
|------|------|
| **取得する情報** | 駐車スポットの位置情報（緯度・経度・住所）、レビュー（テキスト・写真）、Good/Bad評価 |
| **取得方法** | ユーザーによるアプリ内での入力・投稿操作 |
| **利用目的** | 他のユーザーへの駐車スポット情報およびレビュー情報の共有、アプリのコンテンツ品質の維持・向上 |
| **保存場所** | Firebase Firestore（クラウドデータベース）および端末内SQLiteデータベース |
| **任意/必須** | 任意（コンテンツの投稿はユーザーの任意です） |

### 3.5 アクティビティログ

| 項目 | 内容 |
|------|------|
| **取得する情報** | デバイスID、アクセス日付、利用プラットフォーム（iOS/Android） |
| **取得方法** | アプリ起動時に1日1回、自動的にFirestoreのuser_activityコレクションに記録 |
| **利用目的** | アプリの利用状況の統計分析（DAU/WAU/MAU等）、サービスの改善 |
| **保存場所** | Firebase Firestore |
| **任意/必須** | 自動（アプリ利用に伴い自動的に記録されます） |

## 4. 情報の利用目的

当社は、取得した情報を以下の目的で利用します。

1. **本アプリの機能提供** — 駐車スポットの検索・表示・共有機能の実現
2. **サービスの改善** — 利用状況の統計分析に基づくUI/UXの改善、新機能の検討
3. **コンテンツの品質管理** — 不適切なコンテンツの検出・対応、スポット情報の正確性の維持
4. **利用状況の把握** — DAU/WAU/MAU等の集計によるサービスの運用状況の把握
5. **ユーザーサポート** — お問い合わせへの対応、障害の調査・解決
6. **法令の遵守** — 法令に基づく情報開示への対応

当社は、上記の利用目的の達成に必要な範囲を超えて、ユーザーの情報を利用いたしません。

## 5. 第三者提供

当社は、以下の場合を除き、ユーザーの情報を第三者に提供いたしません。

1. ユーザーの同意がある場合
2. 法令に基づく場合
3. 人の生命、身体または財産の保護のために必要がある場合であって、本人の同意を得ることが困難であるとき
4. 公衆衛生の向上または児童の健全な育成の推進のために特に必要がある場合であって、本人の同意を得ることが困難であるとき
5. 国の機関もしくは地方公共団体またはその委託を受けた者が法令の定める事務を遂行することに対して協力する必要がある場合であって、本人の同意を得ることにより当該事務の遂行に支障を及ぼすおそれがあるとき

## 7. データの保管・オフラインキャッシュ

1. 本アプリは、Firebase Firestoreの永続的ローカルキャッシュ機能（`persistentLocalCache`）を利用しています。これにより、一度取得した駐車スポットデータが端末内にキャッシュとして保存され、オフライン時にも表示が可能となります。
2. 当該キャッシュデータは、Firebase SDKにより端末のローカルストレージに自動的に保存・管理されます。
3. キャッシュデータの削除を希望する場合は、端末の設定からアプリのデータを消去するか、アプリをアンインストールしてください。

## 8. データの保存期間

当社は、取得した情報を以下の期間保存します。

| 情報の種類 | 保存期間 |
|-----------|---------|
| デバイスID（端末内） | アプリのアンインストールまたはデータ消去まで |
| ニックネーム（端末内） | アプリのアンインストールまたはデータ消去まで |
| ユーザー投稿コンテンツ（Firestore） | ユーザーによる削除依頼または当社の判断により削除するまで |
| アクティビティログ（Firestore） | 取得から最大2年間（統計処理後、個別データは削除） |
| ローカルキャッシュ | アプリのアンインストールまたはデータ消去まで |

保存期間の満了後、または利用目的の達成後、当社は速やかに情報を削除または匿名化します。

## 10. 位置情報の取扱い

1. 本アプリは、端末のGPS機能を利用して位置情報を取得します。
2. 位置情報の取得は、OSの位置情報許可ダイアログを通じたユーザーの明示的な許可（オプトイン）に基づいて行われます。
3. 取得した位置情報は、周辺の駐車スポットを検索・表示するというアプリのコア機能の実現にのみ使用されます。
4. ユーザーの現在地の位置情報は端末内でのみ使用され、当社のサーバーには送信・保存されません。
5. ユーザーは、端末の設定からいつでも位置情報の利用許可を取り消すことができます。
