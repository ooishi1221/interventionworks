## 「誰の価値観だ？」— Slight の答え

| 問い | 答え |
|---|---|
| 誰の価値観か？ | 都市ライダー。バイクと身体感覚で繋がりたい層。ゴツい安全装備への違和感を持っている人 |
| 顧客の身体感覚 | 走行中、グローブの厚みが操作感を殺す違和感。手を温めても運転が下手になっては本末転倒 |
| 「報告」を押し付けていないか | 機能を競うのではなく、**触覚を取り戻す**。物理スイッチ・複雑な温度調節 UI は引き算 |
| 存在証明 | 着けてる時に「つけてる感じがしない」が成功 |

## D2C PoC としての位置づけ

Slight 単体の成功 / 失敗を超えて、**ウィットワンの D2C 伴走コンサル営業エビデンス**を作るのが本当の狙い。

- 企画 → ファブレス製造 → CF 販売 → ローンチ後運用 を一気通貫で記録
- 成功 / 失敗 どちらでも「学んだこと」をパッケージ化
- KUROKO の AI Ops 受託と並ぶ、もう一つの**ウィットワンの新事業ピッチ素材**

---

## 既知の戦略課題

- **市場規模が小さい**（電熱グローブ × 美学にこだわる層）→ ニッチ狙いで割り切る
- **5,000円 × 300個 = 150万円**売上では会社事業として小さい → PoC として割り切る
- **CF 凍結条件が厳しい**（14 日 50%）→ 撤退ラインを早めに引いた設計、深追いしない
